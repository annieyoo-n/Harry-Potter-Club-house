# Harry-Potter-Club-house
**Harry Potter Club-House**

Python desktop app developed using tkinter that enables the user to learn more about the Harry Potter world. It retrieves live information on Harry Potter API to show character information, lore, and spells.

**Features**
Browse Characters: Browse all the characters, sort by role (Students/Staff), or sort by Hogwarts House (Gryffindor, Slytherin, Hufflepuff, Ravenclaw).
Spell Book: See a full list of spells and their definitions.
Detailed View: When any name is clicked on, the detailed information will include:
Core, Length, Wood specifications.
Physical characteristics (colour of eyes, colour of hair)
Life (date of birth, ancestry, actor)
Image of the characters was loaded dynamically via the web.
Error Handling: Popups provide easy provisions to connection failure or blank selection.

**Usage Guide**
Left Panel (Browse Options): click the buttons in order to retrieve data. All characters, just students, just staff, or a particular house can all be found in the dropdown menu. You may also change to watching Spells.
List Results: It will be added to the list on the Results box once it is loaded with data.
Right Panel (Details): When a name is selected a name on the list, the details and picture of that name appear in the right panel.
